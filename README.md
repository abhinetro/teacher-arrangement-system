# Teacher Arrangement System

An open-source web application for schools to manage teacher substitutions and arrangements when teachers are absent.

## Features

- Teacher Management
- Class Management
- Timetable Management
- Absence Tracking
- Automatic Arrangement Generation
- PDF Export
- Open Source

## Tech Stack

Frontend:
- React
- TypeScript
- Material UI

Backend:
- Node.js
- Express
- PostgreSQL
- Prisma
